\section{Conclusion}

We presented BiTempQA, a diagnostic benchmark for evaluating bitemporal reasoning, that is, reasoning about \emph{when events occurred} versus \emph{when the system learned about them}, in LLM agent memory systems.
Our evaluation of five memory backends yields three main findings.

First, our results indicate that record-time reasoning is more difficult than event-time reasoning ($p = 0.023$), supporting the usefulness of the dual-timestamp distinction.
Second, retrieval appears to be a major bottleneck ($\sim 55\%$ of sampled failures), suggesting that temporally aware retrieval mechanisms may be a productive direction for improvement.
Third, the evaluated memory backends are complementary: an oracle ensemble reaches 85.4\% (+9.8pp), and graph-based memory shows strong adversarial robustness (97.5\%) despite lower temporal accuracy.

% 中文注释：这里只保留正常正文字体，不再额外强化语气。
Taken together, these findings suggest several directions for temporally aware memory systems, including bitemporal indexing, hybrid retrieval architectures that combine vector breadth with graph precision, and external verification beyond sole reliance on internal confidence.

Future work includes human expert validation, English translation, evaluation of production memory systems (e.g., Mem0, Graphiti, and TMG), and extension to longer scenarios with richer temporal dynamics.
